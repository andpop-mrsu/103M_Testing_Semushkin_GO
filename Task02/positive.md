# Чек-лист: Позитивные сценарии

## Игровая механика

| № | Сценарий | Ожидание |
|---|----------|-----------|
| 1 | Левый клик по клетке с числом (1–8) | Клетка раскрывается, отображается соответствующая цифра |
| 2 | Левый клик по пустой клетке (0 соседей) | Клетка открывается, автоматически раскрываются все смежные пустые и граничащие с ними клетки с числами |
| 3 | Первое нажатие в любом месте поля | Клетка гарантированно безопасна, в радиусе 1 клетки от неё нет мин |
| 4 | Клик по клетке с миной | Наступает проигрыш, все мины становятся видны, ошибочная клетка выделяется |
| 5 | Открытие всех клеток без мин | Игра завершается победой, выводится соответствующее сообщение |
| 6 | Установка флага (правый клик) | На закрытой клетке появляется флаг |
| 7 | Снятие флага (правый клик) | Флаг исчезает, клетка снова закрыта |
| 8 | Установка флага долгим нажатием (мобильное устройство) | Через 400+ мс на клетке появляется флаг |
| 9 | Снятие флага долгим нажатием | Флаг убирается |
| 10 | Короткое касание по клетке (мобильное) | Клетка открывается (аналогично левому клику) |
| 11 | Автооткрытие пустых клеток с флагами | Клетки с флагами не раскрываются рекурсивно |
| 12 | Победа без расстановки всех флагов | Достаточно открыть все безопасные клетки, ставить флаги необязательно |

## Уровни сложности

| № | Сценарий | Ожидание |
|---|----------|-----------|
| 13 | Выбор «Новичок» | Поле 8×8, 10 мин |
| 14 | Выбор «Любитель» | Поле 16×16, 40 мин |
| 15 | Выбор «Профессионал» | Поле 16×30, 99 мин |
| 16 | Нажатие «Новая игра» в процессе | Генерируется новое поле с текущими параметрами |
| 17 | Смена уровня после завершения партии | Открывается меню выбора, новая игра начинается с выбранными параметрами |

## Таймер

| № | Сценарий | Ожидание |
|---|----------|-----------|
| 18 | До первого хода | Таймер на `00:00`, не запущен |
| 19 | После первого хода | Таймер начинает отсчёт в формате `mm:ss` |
| 20 | При победе | Таймер останавливается, итоговое время показывается на экране |
| 21 | При поражении | Таймер останавливается |
| 22 | Нажатие «Пауза» | Таймер приостанавливается, появляется экран паузы |
| 23 | Нажатие «Продолжить» | Таймер возобновляется с момента остановки |
| 24 | Превышение лимита времени (10 мин для «Новичка») | Игра автоматически заканчивается поражением |
| 25 | Превышение лимита (40 мин для «Любителя») | Аналогично |
| 26 | Превышение лимита (99 мин для «Профессионала») | Аналогично |

## Сохранение и загрузка

| № | Сценарий | Ожидание |
|---|----------|-----------|
| 27 | Автосохранение после каждого хода | В `localStorage` сохраняется поле, статусы клеток, время, номер хода |
| 28 | Перезагрузка страницы при незавершённой игре | Появляется предложение продолжить |
| 29 | Согласие продолжить | Игра восстанавливается полностью |
| 30 | Ручное сохранение через кнопку | Открывается диалог ввода имени |
| 31 | Ввод имени и подтверждение | Игра сохраняется в `localStorage` под указанным именем |
| 32 | Открытие экрана загрузки | Отображается список всех сохранённых игр |
| 33 | Выбор и загрузка сохранения | Состояние игры полностью восстанавливается |
| 34 | Нажатие на иконку корзины у сохранения | Появляется попап с кнопками «Отмена» и «Удалить» |
| 35 | Подтверждение удаления | Сохранение исчезает из списка и `localStorage` |
| 36 | Отмена удаления | Сохранение остаётся, попап закрывается |

## Статистика

| № | Сценарий | Ожидание |
|---|----------|-----------|
| 41 | Открытие панели статистики | Показывается общая информация: всего игр, побед, поражений, процент побед |
| 42 | Статистика по уровням | Для каждого уровня — количество побед и лучшее время |
| 43 | После победы | Количество побед растёт, лучшее время обновляется |
| 44 | После поражения | Количество поражений увеличивается |
| 45 | Отображение даты последней игры | В зависимости от локали: `dd.mm.yyyy` (ru) или `mm/dd/yyyy` (en) |
| 46 | Очистка статистики | После подтверждения все данные сбрасываются |
