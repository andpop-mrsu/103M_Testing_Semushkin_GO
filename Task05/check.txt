--------------------------------------------------
ПОЗИТИВНЫЕ ТЕСТЫ (корректные входные данные)
--------------------------------------------------

1. Равносторонний треугольник
Входные данные: a=3, b=3, c=3
Ожидаемый результат: "equilateral"
Описание: проверка случая, когда все стороны равны

2. Равнобедренный треугольник (две стороны равны)
Входные данные: a=3, b=4, c=3
Ожидаемый результат: "isosceles"
Описание: проверка случая с двумя равными сторонами

3. Разносторонний треугольник
Входные данные: a=3, b=4, c=5
Ожидаемый результат: "nonequilateral"
Описание: все стороны различны

4. Равнобедренный (другая комбинация)
Входные данные: a=5, b=5, c=8
Ожидаемый результат: "isosceles"
Описание: проверка другой комбинации равных сторон

5. Дробные значения сторон
Входные данные: a=2.5, b=2.5, c=3.0
Ожидаемый результат: "isosceles"
Описание: проверка работы с float

--------------------------------------------------
НЕГАТИВНЫЕ ТЕСТЫ (ошибочные входные данные)
--------------------------------------------------

6. Нарушение неравенства треугольника
Входные данные: a=1, b=2, c=3
Ожидаемый результат: исключение IncorrectTriangleSides
Описание: сумма двух сторон равна третьей → треугольник не существует

7. Сумма двух сторон меньше третьей
Входные данные: a=1, b=2, c=4
Ожидаемый результат: исключение IncorrectTriangleSides
Описание: невозможный треугольник

8. Отрицательная сторона
Входные данные: a=-1, b=2, c=3
Ожидаемый результат: исключение IncorrectTriangleSides
Описание: длина стороны не может быть отрицательной

9. Нулевая сторона
Входные данные: a=0, b=2, c=3
Ожидаемый результат: исключение IncorrectTriangleSides
Описание: сторона не может быть равна нулю

10. Некорректный тип данных (строка)
Входные данные: a="a", b=2, c=3
Ожидаемый результат: исключение IncorrectTriangleSides
Описание: стороны должны быть числами

11. Некорректный тип данных (None)
Входные данные: a=None, b=2, c=3
Ожидаемый результат: исключение IncorrectTriangleSides
Описание: проверка на отсутствие значения

12. Все стороны равны нулю
Входные данные: a=0, b=0, c=0
Ожидаемый результат: исключение IncorrectTriangleSides
Описание: граничный случай некорректных данных
